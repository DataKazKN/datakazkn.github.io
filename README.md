# datakazkn.github.io

Personal site / portfolio / blog index for [@DataKazKN](https://github.com/DataKazKN).

Live at: **https://datakazkn.github.io**

## What's here

- Single-page static site (zero dependencies, no build step)
- Hero section + Apify actors showcase + articles index + contact
- Hosted free on GitHub Pages

## Deploy

```bash
git push origin main
# GitHub Pages auto-deploys from main branch within ~30s
```

Settings: Repository → Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `/ (root)`.

## Stack

- Plain HTML/CSS, served by GitHub Pages
- Google Fonts: Bricolage Grotesque · Inter · JetBrains Mono
- Color palette: Apify-inspired dark indigo + teal/green accents
